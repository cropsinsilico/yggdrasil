
Welcome to the 2021 Crops in Silico Hackathon!
##############################################

These materials walk through some of the basics to using |yggdrasil| to connect models using Jupyter notebooks. These exercises were created for the 2021 Crops in Silico hackathon, but can be used by anyone who would like to learn how to use |yggdrasil|. The 2021 Crops in Silico hackathon was recorded and can be viewed `here <https://uofi.app.box.com/s/kjhahvhvp9dss7u75wycxh7vymnlqirr>`_.
