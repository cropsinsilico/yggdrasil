DebuggingÂ¶

For those participating in the CiS 2021 Hackathon in June 2021 and those referencing these materials later, we recommend the following strategies for debugging issues you have accessing the notebooks.

  1. If you are trying the MyBinder set up and are using a browser other than Mozilla Firefox, Google Chrome, or Safari, install one of those and try again

  2. If it is an error in a notebook cell, try restarting and running all of the cells (in case previous cells were missed).

  3. Check for similar issues that have been reported as issues here including the issues that have been closed.

  4. If you are having issues with a local install, check this page for known errors and their solutions.

  5. If your issue has not been reported, open a new one here.
